# Dinesh's Personal Landing Page

A clean, modern personal landing page built with **Google Material You (MD3)** design principles.

## 🚀 GitHub Pages Deployment

To host this page on GitHub Pages, follow these simple steps:

1.  **Create a New Repository**: Go to GitHub and create a new public repository (e.g., `portfolio`).
2.  **Upload Files**: Upload `index.html` and `style.css` to the root of your repository.
3.  **Enable GitHub Pages**:
    - Go to **Settings** > **Pages**.
    - Under "Build and deployment", set the source to **"Deploy from a branch"**.
    - Select your branch (usually `main`) and the folder `/ (root)`.
    - Click **Save**.
4.  **View Your Site**: Within a few minutes, your site will be live at `https://<your-username>.github.io/<repository-name>/`.
